# Sentinella Glaciale

Applicazione per il monitoraggio dei ghiacciai

## Requisiti
- Android Studio (ultima versione consigliata)
- JDK 17
- Gradle incluso nel progetto

## Installazione
1. Estrarre il file ZIP
2. Aprire Android Studio
3. Selezionare "Open an existing project"
4. Selezionare la cartella Sentinella_glaciale
5. Attendere il download delle dipendenze Gradle

## Esecuzione
- Collegare un dispositivo Android oppure avviare un emulatore
- Premere il pulsante "Run"
